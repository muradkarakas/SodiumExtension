◆ isInPageReady

var isInPageReady

This variable is used to check whether we are called during "document ready" phase or after that time. This is used for executint response actions which are generated by native or HT/SQL function call during GET request. For this reason, initPage checks the value of these variable to execute GET response actions before delete them.
