Modifies the current value of the TAGInput's "value" property in a TAGDataBlock.

Parameters
currentMKSession
dataBlockNameDatablock name of the TAGInput instance belongs to.
pPropNameProperty name of the TAGInput whose value will be set
pPropValueNew value to be set
Returns
char* Returns 0 if finds the block specified by dataBlockName and property specified by pPropName parameter. If error occurs, returns the reason as int value.
0 -> success.
1 -> TAGInput not found

"valueState" proeprty of the TAGInput will be set to INPUT_TAG_VALUE_DIRTY.
