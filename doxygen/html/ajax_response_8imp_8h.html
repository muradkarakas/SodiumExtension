Detailed Description

This file contains C functions which implements HT/SQL built-in functions.


C Developer should take care of these request types :

Request can be one of these:

  • native call or
  • HT/SQL function call

Request can be one of these:

  • GET (page request) or
  • POST (Ajax request)

    C functions should act accordingly and produce different result set for request types.

    C functions can check the request type using these code block in their code.

    if (currentMKSession->currentRequestType == REQUEST_TYPE_GET) { page request } else { ajax request }



    Example implementation is __sendMessageResponse().

    Function declarations are found in the different part of the file according to their Native or HT/SQL implementation category (have look at the source file).
    Native call requests should be handled by a function whose name contains "Native" word like __sendRefreshBlockNativeResponse().
