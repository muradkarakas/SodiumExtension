Detailed Description

Deals with AJAX request.

AJAX requests are categorized by their types which can be either HT/SQL function Call or Native C Code Call.
Type information is received as HTTP POST request form parameter.
Data structure for both request type are as follows :

  • HT/SQL function name should/will be in the form name value:
    htsqlproceduretorun=

  • Native C function name should/will be in the form name value:
    nativeproceduretorun=
