This C function runs "on_logon" HT/SQL trigger function if exists.

Parameters
currentMKSessionMKSession*
Returns
void

If an HT/SQL application needs a database connection (TAGConnection instance) but there is no one available, "on_logon" HT/SQL trigger is called to notify application to create a new one. HT/SQL programmer should create a new connection and corresponding TAGConnection instance by using the command below in that trigger :

void on_logon() {
create_connection("yugs.world", "ORCL.WORLD", "hr", "hr");
}
