If an error is occurred: Creates a new log record, Set the clientMessage property of the currentMKSession instance. So, before calling, clientMessage should be clear by calling clearClientMessage(MKSession *currentMKSession). Sends the Oracle error message string to the browser as plain text.
Caller may use code block below to get error message.

clearClientMessage(currentMKSession);
excute mkOCI_Execute_Stmt(....); char *msg = getClientMessage(currentMKSession); if (msg) { __sendErrorResponse(currentMKSession, "Veritabanýna yazýlamadý", msg); mkFree(currentMKSession->heapHandle, msg); }
