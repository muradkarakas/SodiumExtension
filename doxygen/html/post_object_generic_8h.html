Detailed Description

Contains functions and data structures for creating, finding, destroying of post phase variables.

Post phase variables are the local variable. They are created before executing HT/SQL or Native function call from the browser and destroyed after HT/SQL or Native function execution ended.

Typedef Documentation

◆ PostObjectGeneric

typedef struct _PostObjectGeneric PostObjectGeneric

PostObjectGeneric data type is a linked list type. Each PostObjectGeneric instance stores post phase/local variable and point another PostObjectGeneric instance for another variable. Post phase/local variables can be any one of the predefined types. Allowed types are stored in each PostObjectGeneric instance variable declared as is

POST_OBJECT_TYPE postObjectType



There is no need to define a new object type for phase/local variable variables. So, PreObjectVariable variable type is used for pre and post variables.
