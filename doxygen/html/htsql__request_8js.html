◆ sendAJAXRequest()

function sendAJAXRequest ( currentSelection  )

Send AJAX request to the server.

What type of request will be sent to the server is firstly determined by the "name" attribute of the html element. Then the type of the html element's value will be checked.

According to value of "name" and "type" attribute of the html element, either "native" or "htsql" procedure call is made.
